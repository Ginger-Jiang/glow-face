# 焕脸
  
  这是一个 Serverless 项目。

## 页面

- 首页
  - 轮播图
  - 功能模块
  - 广告嵌入

- 智能融脸
- 动漫照片
- 智能美妆
- 智能美图

## 目录结构

```
. 焕脸
├── api                   # 请求封装
├── common                # 公共
├── components            # 公共组件
├── img                   # 图片资源
│   ├── common            # 全局图片资源
│   └── home              # 首页图片
└── pages                 # 页面
    ├── cartoon           # 动漫照片
    ├── home              # 首页
    │   ├── banner        # 首页轮播图模块
    │   └── modules       # 首页功能模块
    └── melting           # 融脸
```
