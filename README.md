# 焕脸

<h2>一个基于 Serverless 架构, 使用 cloudBase 实现服务的字节跳动小程序.<h2>

# 技术关键词

- Serverless
- cloudBase
- Egg
- 小程序
- 云函数
- 云数据库
- 云存储

# 功能

- 人脸搜索(规划中)
- 融脸(规划中)
- 未完待续...

# 开发

请先阅读云开发[官方文档]('https://docs.cloudbase.net/quick-start/create-env.html')

## 后端服务
```
yarn install
yarn run dev
```

## 小程序
```
直接使用字节跳动小程序打开即可
```

## 部署
```
tcb
```
